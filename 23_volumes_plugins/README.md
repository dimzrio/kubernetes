
# Volume Plugins #
Kubernetes have many volume plugins.

### Remote Storage 
- GCE Persistent Disk (GCE-PD)
- AWS Elastic Block Storage
- Azure File Storage
- iSCSI
- NFS
- Ceph
- GlusterFS

### Ephemeral Storage
- EmptyDir
- Expose Kubernetes API (Secret & ConfigMap)

### Local Persistent Volume

### Out-of-Tree
- Flex (Exec a binary)
- CSI (Container Storage Interface)

### Other
- Host Path
